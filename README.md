# 🏥 Samrijo Hospital Website

A modern, responsive, and conversion-focused hospital website built using **HTML, Tailwind CSS, and JavaScript**.  
Designed to create trust, deliver a premium medical experience, and drive patient inquiries via WhatsApp and direct calls.

---

## 🚀 Overview

Samrijo Hospital, located in **Burari, Delhi**, provides advanced laser and laparoscopic surgical treatments with a strong focus on precision, faster recovery, and patient-first care.

This website is built to match the quality and structure of top hospital brands like Apollo and Max Healthcare.

---

## ✨ Key Features

- Fully responsive (Mobile, Tablet, Desktop)
- Premium medical UI (Blue & White theme)
- Sticky navigation bar
- WhatsApp integration with dynamic service messages
- One-click call functionality
- Clean and structured layout
- SEO optimized
- Lightweight and fast loading
